# 万能小工具库（完善中）
## 一.🎯 五子棋 · 人机对战
一个纯前端、开箱即用的五子棋游戏，支持三种难度和先后手选择，AI 基于智能评分算法，兼顾进攻与防守。

🎮 玩法说明
  三种难度：简单、中等、困难，适合不同水平的玩家
  自由选择先后手：玩家可执黑（先手）或执白（后手）
  智能 AI：基于连子数与开放性的评分算法，具备基础攻防策略
  响应式设计：在电脑和手机上都可流畅游玩
  点击棋盘落子，AI 自动响应

🖥️ 技术栈
HTML5
CSS3（自定义变量 + 响应式布局）
原生 JavaScript（无框架依赖）

📦 运行方式
本地运行
直接双击 [gomoku_standalone.html](gomoku_standalone.html) 文件，在浏览器中打开即可开始游戏。
在线预览
将 gomoku_standalone.html 部署到任意静态托管服务（如 GitHub Pages、Vercel、Netlify）即可在线游玩。

## 简易网络爬虫
一个轻量级的 Python 网页爬虫脚本[scraper.py](scraper.py)，可提取网页标题、正文预览和所有链接，适合快速内容抓取与学习测试。
📦 功能
✅ 提取网页标题
✅ 获取正文预览（前500字符）
✅ 提取并去重所有链接（显示前30个）
✅ 自动处理相对路径为绝对 URL
✅ 设置合理超时和请求头，降低被屏蔽风险
⚠️ 注意事项
⚠️ 仅限学习与合法用途，请遵守目标网站的 robots.txt
🕐 请勿高频请求，建议加入适当延迟，尊重服务器资源
